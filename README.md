## 🛠️ Installation & Setup

1. Clone the repository
```bash
git clone https://github.com/stylinn-mansi/echelon-reach.git
```

2. Install dependencies
```bash
cd echelon-reach
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
